Quick Add for Followish (unofficial)

Политика конфиденциальности · обновлено 3 июля 2026

Это неофициальное расширение для браузера, помогающее добавлять товары в вишлисты сервиса Followish в один клик. Расширение не связано с Followish и разработано независимо.

Какие данные обрабатываются

  • Токен авторизации Followish. Когда вы открываете followish.io залогиненным, расширение копирует ваш токен сессии из локального хранилища сайта в локальное хранилище расширения (chrome.storage.local) на вашем устройстве. Токен нужен только для обращения к API Followish от вашего имени. Он никуда больше не передаётся и не покидает ваш браузер, кроме запросов к core.followish.io.
  • Данные товара с открытой вкладки. Только когда вы нажимаете на иконку расширения, оно считывает со страницы название, цену, изображения и ссылку, чтобы заполнить форму. Ничего не читается в фоне.

Куда уходят данные

Только на core.followish.io — чтобы создать подарок в вашем вишлисте, когда вы нажимаете «Добавить». Больше никуда.

Чего расширение НЕ делает

  • не собирает аналитику и не отслеживает вас;
  • не передаёт данные разработчику или третьим лицам;
  • не продаёт данные;
  • не имеет собственных серверов — работает только между вашим браузером и Followish.

Разрешения

  • storage — хранить токен и выбранный вишлист локально.
  • activeTab + scripting — прочитать данные товара с текущей вкладки в момент нажатия.
  • доступ к followish.io и core.followish.io — синхронизация токена и вызовы API.
