# Followish Quick Add

Chrome-расширение (Manifest V3): добавляет товар с любой открытой страницы в
вишлист [Followish](https://followish.io) в один клик. Работает на маркетплейсах
с антиботом (Ozon, Wildberries) — потому что читает **уже отрендеренную** страницу
в твоём браузере, а не парсит её снаружи.

## Как это работает

- **`token-sync.js`** — контент-скрипт на `followish.io`. Когда ты залогинен,
  копирует `localStorage.token` в хранилище расширения. Токен подхватывается сам.
- **`popup.js`** — при клике на иконку инжектит экстрактор в активную вкладку,
  вытаскивает название / цену / картинку / ссылку (JSON-LD → OG-теги → эвристики),
  даёт всё поправить и выбрать вишлист.
- **`background.js`** — единственное место, что ходит в API
  `core.followish.io/api`. Создаёт подарок:
  `POST /wishlists/{linkKey}/presents` (ключ картинки — `imageLink`).
- **`options.html`** — статус токена + ручная вставка, если авто-синк не сработал.

## Установка (dev)

1. `chrome://extensions` → включи **Режим разработчика**.
2. **Загрузить распакованное расширение** → выбери эту папку.
3. Открой `followish.io/app`, залогинься (токен синкнется).
4. На странице товара нажми иконку расширения → проверь поля → **Добавить**.

По умолчанию льёт в вишлист **«Инбокс»** (можно сменить в попапе — выбор запоминается).

## Нюансы парсинга

- **Tilda / brand-магазины** — JSON-LD или OG обычно есть, цена варианта берётся
  из выбранного оффера. Если на странице несколько товаров-виджетов, проверь цену
  перед сохранением.
- **Ozon / WB** — антибот-капчу проходишь ты (страница уже открыта), расширение
  просто читает DOM. Скрапинг снаружи там невозможен.
- Любое поле правится в попапе перед сохранением — экстрактор не обязан быть идеальным.

## Горячие клавиши

- `Enter` в поле — сохранить.
- `⌘/Ctrl + Enter` — сохранить из любого места попапа.
- `Esc` — закрыть.

## Варианты и картинки

- Если на странице несколько редакций (Tilda-табы, side-by-side) — вверху попапа
  чипсы вариантов. Тапнул — подставились имя + цена + галерея этой редакции.
- Лента миниатюр — все годные фото со страницы; клик выбирает картинку.

## Иконки

`icons/icon.svg` — исходник (фиолетовая 4-лучевая искра). PNG (16/48/128)
генерятся из него: `magick -background none icons/icon.svg -resize NxN icons/iconN.png`.
Тема попапа — светлая + тёмная (по `prefers-color-scheme`).

## Безопасность

Токен — это доступ к твоему аккаунту. Он живёт только в `chrome.storage.local`
на твоей машине, **в репозиторий не попадает** (`.gitignore`). Не коммить его руками.
