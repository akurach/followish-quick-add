# Chrome Web Store — материалы листинга

Всё готово для заливки. Скопируй поля ниже в Developer Dashboard.

## Основное

- **Название:** `Quick Add for Followish (unofficial)`
- **Краткое описание (summary, ≤132):**
  `В один клик добавляет товар с любой страницы в вишлист Followish — имя, цена, галерея и варианты.`
- **Категория:** Shopping
- **Язык:** Русский
- **Иконка:** `icons/icon128.png` (в zip)

## Подробное описание

```
Quick Add for Followish — неофициальное расширение для быстрого наполнения вишлистов Followish.

Открыл товар в любом магазине → нажал иконку → товар в вишлисте. Название, цена, картинки и варианты подтягиваются автоматически, всё можно поправить перед сохранением.

Возможности:
• Один клик с любой страницы — маркетплейсы, бренд-магазины, книжные, авто.
• Автозаполнение: имя, цена, ссылка.
• Пикер картинок — выбираешь нужное фото из всех на странице.
• Пикер вариантов — если у товара несколько редакций/комплектаций.
• Выбор вишлиста, запоминание последнего.
• Светлая и тёмная тема, горячие клавиши (Enter / ⌘↵ — сохранить).

Токен авторизации подхватывается автоматически, когда вы залогинены на followish.io. Данные не собираются и не передаются третьим лицам — работает только между вашим браузером и Followish.

Расширение неофициальное и не связано с сервисом Followish.
```

## Single purpose (единственное назначение)

```
Добавление товара с текущей веб-страницы в вишлист пользователя на followish.io.
```

## Обоснование разрешений (permission justification)

- **storage** — хранить локально токен авторизации и выбранный по умолчанию вишлист.
- **activeTab** — доступ к активной вкладке в момент нажатия иконки, чтобы прочитать данные товара.
- **scripting** — выполнить скрипт-экстрактор на текущей странице для чтения имени/цены/картинок.
- **host: followish.io** — синхронизация токена сессии.
- **host: core.followish.io** — вызовы API для создания подарка в вашем вишлисте.

## Privacy

- **Политика конфиденциальности (URL):** https://akurach.github.io/followish-quick-add/privacy.html
- **Data usage / раскрытие данных в форме стора:**
  - Собираемые данные: «Authentication information» (токен) — **хранится локально**, используется только для работы функции, **не продаётся, не передаётся третьим лицам**.
  - Разработчик не собирает и не получает пользовательские данные.
  - Отметить: данные не используются для целей, не связанных с основной функцией; не продаются; не используются для оценки кредитоспособности.

## Как залить (для тебя)

1. https://chrome.google.com/webstore/devconsole → один раз оплатить $5 (аккаунт разработчика).
2. New item → загрузить `followish-quick-add.zip`.
3. Заполнить поля выше, добавить скриншоты, указать URL политики.
4. Visibility: **Public** (или Unlisted — по ссылке).
5. Submit → ревью Google ~1–3 дня.
